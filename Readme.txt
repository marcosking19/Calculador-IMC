¡Hola! Bienvenido/a a la Calculadora de IMC desarrollada como parte de un proyecto académico.

Descripción básica:
Este programa es una calculadora de Índice de Masa Corporal (IMC) desarrollada en Python utilizando la biblioteca 
Tkinter para la interfaz gráfica. Permite a los usuarios ingresar su nombre, peso, altura, edad y sexo para calcular 
su IMC y determinar su categoría (bajo peso, peso normal, sobrepeso, obesidad). Los resultados se pueden guardar en un 
archivo CSV para su posterior revisión.

Características de la versión:
- Interfaz gráfica amigable con Tkinter.
- Cálculo del IMC ajustado por edad y sexo.
- Visualización de categorías de IMC.
- Opción para guardar los datos del usuario en un archivo CSV.
- Imagen de fondo personalizable.
